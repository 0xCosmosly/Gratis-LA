# Gratis LA ☕

**[🌟 View the Live Site](https://0xcosmosly.github.io/Gratis-LA/)**

Tired of surprise fees and tipping math at the end of your meal? Gratis LA is a community-sourced map of Los Angeles restaurants that either enforce a strict no-tipping policy or transparently fold gratuity and service fees into their menu prices. 

This project aims to bring clarity to an increasingly confusing dining landscape by helping locals find straightforward, transparent dining experiences.

## Features that make it great

- **Interactive Map & List**: Explore restaurants via a beautiful Leaflet-powered map or a detailed list view. It works seamlessly across desktop and mobile.
- **Strict Verification**: We don't guess. Every single listing requires hard proof (official menus, newsroom pages, or hospitality PDFs) showing that tips are not expected, or fees are clearly included.
- **Smart Filters**: Quickly filter by `No Tip` spots, `Included` fee locations, or by dining style (Sit-Down, Quick Service, Bars).
- **Location Aware**: Grant location access to instantly find the transparent pricing spots closest to you.
- **Community Moderation**: Users can submit new leads and cast verification votes to maintain the map's high standards.

## Under the Hood

Gratis LA is built for speed and reliability, whether you're at home or out on the town.

- **Frontend**: A lightning-fast React Single Page Application (SPA) built with Vite.
- **Backend**: Powered by Supabase for live data syncing and community voting.
- **PWA Ready**: Install it directly to your phone's home screen for app-like access on the go.
- **Fallback Mode**: Designed to work in a local "demo" mode with bundled data if the cloud connection is ever unavailable.
